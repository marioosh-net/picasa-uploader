picasa-uploader
===============

Picasa Web Albums Console Uploader


usage: java -jar picasa-uploader.jar [options] <dir1|file1>
            <dir2|file2>...
 -d <arg>   album description
 -h         help
 -l         list albums
 -p <arg>   password [REQUIRED]
 -t <arg>   album title
 -u <arg>   user [REQUIRED]
 -v         be verbose
